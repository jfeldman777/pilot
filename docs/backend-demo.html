1. UI (терминал A)

Раздача статики и страниц demo.

Команда
pip install flask python app.py

Откройте эту страницу: backend-demo.html · порт 5001

2. API v2 (терминал B)

FastAPI + pandapower DC screening.

Установка и запуск
pip install -r api/requirements.txt python -m uvicorn api.main:app --port 8000
  • api-demo.html → Health → "pandapower": true
  • Load example → POST /dc-run → "converged": true

3. Batch War-Gaming (терминал C)

Прогоны вне браузера — TSO 100 узлов, до 50k runs.

Быстрый smoke (100 runs)
node scripts/run_mc_batch.mjs 100 42
Демо для заказчика (1000 runs)
node scripts/run_mc_batch.mjs 1000 42

Затем откройте batch-results.html — поля runs, worst_damage, pareto.
